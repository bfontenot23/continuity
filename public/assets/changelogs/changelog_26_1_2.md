# Continuity Version 26.1.2 Changelog

This is a smaller update mostly focused on rounding out the website itself as well as fixing bugs.  Larger changes planned for next update.

## What's New
\+ This app is now officialy using GNU General Public License v3.0.  Contribute to this project on our [Github](https://github.com/bfontenot23/continuity)!
\+ Projects now store the version of the app they were created on as well as a description field.
\+ App info can now be viewed by clicking the info icon (There is also a Ko-fi link there!).
\+ Version changelogs can now be viewed by clicking the clipboard icon.
\+ Project settings (title and description) can now be changed by clicking the settings icon.
\+ Arcs are now draggable in the timeline edit menu.
\- Arc Mode has been removed (Arcs are now visible by default).

## Bug Fixes
- Adjusted versioning scheme to a more modern format of year.month.buildnumber.
- Textbox text no longer renders over the plus menu.
- "New Project" button now has correct formatting.
- Default endpoints on lines are now "None" instead of "Dot".
- Default endpoints on branches are now "None" and "Arrow" instead of "Dot".
- Adjusted UI for smaller screens in preparation of touch support.
- Error handling when creating a project now displays correctly.
- Branches correctly reposition themselves when chapters are added/moved/deleted.
